Welcome to my blog! This is the first post on my new Astro-powered website.

## Why Astro?

Astro is a modern static site generator that focuses on delivering fast websites by shipping **zero JavaScript** to the browser by default. It supports Markdown content out of the box, making it perfect for blogs.

### Key Features

- **Content Collections** — Type-safe Markdown with Zod schema validation
- **File-based Routing** — Pages are just files in the `src/pages/` directory
- **Component Islands** — Add interactivity only where you need it
- **Fast by Default** — No JS shipped unless you opt in
